# Aegis Boundary Skill (extension context)

This extension provides the `aegis-boundary` skill (see `skills/aegis-boundary/SKILL.md`).

Invoke it when work touches Aegis surfaces: aegis-trust SDK or Aegis Core
integrations, README/site/sales claims about Aegis, or any change to what an
AI agent is allowed to see, do, or output through Aegis. It performs an
ADVISORY review and returns READY / REVIEW / STOPPED with machine-check
evidence. It does not authorize, enforce, or produce receipts — runtime
enforcement belongs to the aegis-trust SDK (LITE) and Aegis Core (FULL).
